# AshodsLib
A high powered windows framework for developing applications in C,
hoping to be a replacement for Boost

compiled with Mingw

compiler flags

-std=c99<br>
-Wall<br>
-Wextra<br>
-s<br>

####All optimizations MUST be disabled


The following MingW libraries must be linked..

libwldap32.a<br>
libgdi32.a<br>
libkernel32.a<br> 
libnetapi32.a<br>
libuser32.a<br>
libvfw32.a<br>
libwinmm.a<br>
libws2_32.a<br>
libversion.a<br>
libcrypt32.a<br>
libsetupapi.a<br>
