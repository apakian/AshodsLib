# AshodsLib
A high powered windows framework for developing applications in C,
hoping to be a replacement for Boost

compiled with Mingw

compiler flags

-std=c99
-Wall
-Wextra
-s

All optimizations MUST be disabled

The following MingW libraries must be linked..

libwldap32.a
libgdi32.a
libkernel32.a
libnetapi32.a
libuser32.a
libvfw32.a
libwinmm.a
libws2_32.a
libversion.a
libcrypt32.a
libsetupapi.a
